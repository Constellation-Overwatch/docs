## Getting Started

This repository contains the source code for the [constellation-overwatch.dev](https://constellation-overwatch.dev) documentation site.

### Development

To run the development server locally:

```bash
bun install
bun dev
```

Visit [http://localhost:3000](http://localhost:3000) to preview the site.

## Documentation Content

The documentation content is located in the `content/` directory and written in MDX.

- `content/docs/`: Main documentation pages.
- `public/`: Static assets including images and the `install.sh` script.
