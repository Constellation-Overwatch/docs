# API Reference

Complete documentation for the Constellation Overwatch REST API.

## Authentication

All API requests require authentication using a Bearer token:

```bash
curl -H "Authorization: Bearer your-api-token" \
  http://localhost:8080/api/v1/organizations
```

## Base URL

```
http://localhost:8080/api/v1
```

## Endpoints

### Organizations

#### Create Organization

```bash
POST /api/v1/organizations
```

**Request Body:**
```json
{
  "name": "My Fleet",
  "org_type": "civilian", 
  "description": "My drone fleet operations"
}
```

### Entities

#### Register Entity

```bash
POST /api/v1/entities?org_id={org_id}
```

**Request Body:**
```json
{
  "name": "Drone-001",
  "entity_type": "aircraft_multirotor",
  "description": "Primary inspection drone",
  "metadata": {
    "model": "DJI-M300",
    "serial": "ABC123456"
  }
}
```
