**Open Source C4 Data Fabric for Industrial Edge Computing**

Constellation Overwatch is a lightweight industrial data stack designed with ontological data primitives for UAVs, robotics, sensors, and real-time video intelligence at the edge under your security control.

### Installation

```bash
# Linux / macOS
curl -LsSf https://constellation-overwatch.github.io/overwatch/install.sh | sh

# Windows
powershell -ExecutionPolicy Bypass -c "irm https://constellation-overwatch.github.io/overwatch/install.ps1 | iex"
```

### Core Features

| Feature | Description |
| ------- | ----------- |
| **Real-time Messaging** | Low-latency pub/sub communication powered by NATS JetStream |
| **Telemetry Streaming** | Efficient handling of high-frequency sensor data from edge devices |
| **Multi-Entity Support** | Manage drones, robots, sensors, and autonomous systems |
| **Secure API** | RESTful API with bearer token authentication |
| **Offline First** | Full security perimeter isolation for air-gapped environments |
| **Edge Architecture** | Portable deployment with proximity-based processing |

### Technology Stack

Powered by modern, battle-tested technologies:

- **Go** - High-performance backend with minimal footprint
- **NATS JetStream** - Embedded messaging with global clustering
- **Templ** - Type-safe HTML templating
- **Datastar** - Real-time UI framework
- **Turso** - Embedded SQLite database
